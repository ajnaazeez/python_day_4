# Python Practice - Day 4 

This repository contains my Day 4 Python learning exercises.  
The focus of this day is understanding string manipulation and creating user-defined functions.

##  Topics Covered
- String indexing & slicing
- String methods (upper, lower, replace, split, find, count)
- String concatenation
- Palindrome checking
- Functions in Python
- Function parameters and return values

##  Practice Programs
Some of the tasks included:
- Reverse a string
- Palindrome checker
- Count vowels and consonants
- Word counter
- Simple calculator using functions
- Temperature converter (Celsius to Fahrenheit)
- Greeting function with user input

##  Purpose
The aim of this project is to improve my understanding of strings and functions and learn how to reuse code efficiently using functions.
